# creative-brief-generator
An interactive game design discovery tool to generate structured visual project briefs.

This is a professional discovery tool designed to help game artists and clients sync on project requirements before starting the creative process.

## 🚀 How it works
Instead of long back-and-forth emails, the client fills out this interactive form, selects preferences (art style, platforms, color palettes), and uploads reference images. 

At the end, it generates a **visual summary (PNG)** that can be saved and used as a project roadmap.

## ✨ Features
- 16 specialized steps for game discovery.
- 30 pre-designed color palettes with visual export.
- Image upload support for visual references.
- Mobile-friendly and dark-themed UI.

## 🛠️ Built With
- HTML5 / CSS3
- JavaScript (Vanilla)
- html2canvas library
